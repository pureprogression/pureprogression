# Варианты улучшения Конструктора тренировок

## 🎯 Текущее состояние
- Есть drag & drop
- Есть фильтры по группам мышц
- Можно редактировать подходы/повторения
- Две вкладки: Browse и Selected

## 💡 Варианты улучшения

### Вариант 1: Split-Screen (Рекомендую)
**Концепция:** Разделенный экран - слева библиотека, справа выбранные упражнения

**Преимущества:**
- ✅ Видно все сразу
- ✅ Не нужно переключаться между вкладками
- ✅ Удобно на десктопе
- ✅ На мобильных можно сделать swipe

**Структура:**
```
┌─────────────────────────────────────┐
│  Название тренировки [input]        │
├──────────────┬──────────────────────┤
│              │                      │
│  Библиотека  │  Выбранные           │
│  упражнений  │  упражнения          │
│              │                      │
│  [Фильтры]   │  [Список с drag]     │
│  [Grid/List] │  [Sets/Reps]         │
│              │                      │
└──────────────┴──────────────────────┘
│  [Сохранить] [Отмена]               │
└─────────────────────────────────────┘
```

---

### Вариант 2: Step-by-Step Wizard
**Концепция:** Пошаговый мастер создания

**Преимущества:**
- ✅ Простой и понятный
- ✅ Фокус на одном действии
- ✅ Хорошо для новичков

**Структура:**
```
Шаг 1: Название и описание
Шаг 2: Выбор упражнений
Шаг 3: Настройка подходов/повторений
Шаг 4: Просмотр и сохранение
```

---

### Вариант 3: Card-Based Builder (Apple-style)
**Концепция:** Карточки упражнений, минималистичный дизайн

**Преимущества:**
- ✅ Красиво и современно
- ✅ Интуитивно понятно
- ✅ Хорошо на мобильных

**Структура:**
```
┌─────────────────────────────────────┐
│  Pure Progression                   │
│  Создать тренировку                 │
├─────────────────────────────────────┤
│  [Название]                         │
│                                     │
│  ┌─────────┐  ┌─────────┐          │
│  │ Упр. 1  │  │ Упр. 2  │  [+Добавить]
│  │ 3x12    │  │ 4x10    │          │
│  └─────────┘  └─────────┘          │
│                                     │
│  [Сохранить]                        │
└─────────────────────────────────────┘
```

---

### Вариант 4: Timeline Builder
**Концепция:** Временная шкала тренировки

**Преимущества:**
- ✅ Визуализация последовательности
- ✅ Видно общее время
- ✅ Удобно планировать

**Структура:**
```
Время: 0:00 ──────────────────── 45:00
       │                          │
       ├─ Упр. 1 (5 мин)          │
       ├─ Отдых (1 мин)           │
       ├─ Упр. 2 (5 мин)          │
       └─ ...                     │
```

---

### Вариант 5: Hybrid (Split + Modal)
**Концепция:** Split-screen + модальное окно для деталей

**Преимущества:**
- ✅ Компактный основной экран
- ✅ Детальная настройка в модалке
- ✅ Лучшее использование пространства

---

## 🏆 Мой топ-3:

1. **Вариант 1 (Split-Screen)** - Лучший баланс функциональности и удобства
2. **Вариант 3 (Card-Based)** - Самый красивый, в стиле Apple
3. **Вариант 5 (Hybrid)** - Максимальная функциональность

- ✅ Превью видео при наведении
- ✅ Анимации добавления/удаления
- ✅ Цветовая индикация групп мышц
- ✅ Прогресс-бар создания тренировки

### Функциональные:
- ✅ Быстрый поиск упражнений
- ✅ Шаблоны тренировок
- ✅ Предпросмотр времени тренировки

- ✅ Автосохранение черновика

### UX:
- ✅ Подсказки для новичков
- ✅ Горячие клавиши
- ✅ Undo/Redo
- ✅ Экспорт тренировки
